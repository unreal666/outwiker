ThumbGallery Plugin fügt den Befehl (:thumbgallery:) in die Wiki Natation ein. Der Befehl kann in zwei Modis benutzt werden: Alle Angehängte Dateien anzeigen oder nur ausgewählte Dateien. Syntax (:thumbgallery [params]:) oder (:thumbgallery [params]:) [Attach:]picture_1 [| Comment] [Attach:]picture_2 [| Comment] ... (:thumbgalleryend:) Parameters
  • cols = ... Ein Ganzzahl Wert. Die Anzahl Spalten in der Tabelle. Wenn cols = 0 oder nicht spezifiziert, dann wird die Gallerie nur aus einer Reihe ohne Spalten bestehen.
  • px = ... Ein Ganzzahl Wert. Die Grösse der Vorschaubilder definieren (auf die grössere Seite). Wenn px = 0, oder der Parameter ist nicht definiert, wird der standart aus den Programm Einstellungen genommen.
