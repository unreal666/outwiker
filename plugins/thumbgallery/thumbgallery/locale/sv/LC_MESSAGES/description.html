ThumbGallery-insticket lägger till kommandot (:thumbgallery:) i wiki-noteringen. Detta kommando kan arbeta i två lägen: Visa alla bifogade bilder eller endast valda filer. Syntax (:thumbgallery [parametrar]:) eller (:thumbgallery [parametrar]:) [Attach:]bild_1 [| Kommentar] [Attach:]bild_2 [| Kommentar] ... (:thumbgalleryend:) Parametrar
  • cols = ... Ett heltal. Antal kolumner i tabellen. Om cols = 0 eller ospecificerad, Kommer galleriet att bestå av en miniatyrbildsekvens, utan att delas upp i rader och kolumner.
  • px = ... Ett heltal. Specificerar storleken på miniatyrerna (den största sidan). Om px = 0, eller om parametern är ospecificerad, används standardvärdet från programinställningarna.
