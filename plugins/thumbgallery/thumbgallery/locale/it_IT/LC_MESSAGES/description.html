Il plugin ThumbGallery aggiunge il comando (:thumbgallery:) nella notazione wiki. Questo comando può funzionare in due modalità: mostra tutte le immagini allegate o o solo i file selezionati. Sintassi (:thumbgallery [parametri]:) oppure (:thumbgallery [parametri]:) [Attach:]picture_1 [| Commento] [Attach:]picture_2 [| Commento] ... (:thumbgalleryend:) Parametri
  • cols = ... Un numero intero. Il numero di colonne nella tabella. Se cols = 0 o non viene specificato nessun valore, allora la galleria sarà una sequenza di miniature senza suddivisione in righe e colonne.
  • px =... Un numero intero. Specifica la dimensione delle miniature (sul lato maggiore). Se px = 0, o il parametro non è specificato, il valore predefinito viene ereditato dalle impostazioni del programma.
