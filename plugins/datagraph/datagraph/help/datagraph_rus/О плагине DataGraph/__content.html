Плагин DataGraph предназначен для построения графиков по текстовым данным. Данные могут вводиться непосредственно на викистранице или читаться из прикрепленных текстовых файлов.

Для построения графиков используется команда (:plot:). Более подробно о формате данных, с которыми может работать плагин DataGraph читайте в разделе Формат данных.

Для построения графиков используется библиотека Highcharts, кроме того, плагин DataGraph использует библиотеку python-dateutil.
