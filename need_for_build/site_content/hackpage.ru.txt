!!! [[#using]] Использование плагина

После установки плагина HackPage в главном меню появится новое подменю "HackPage" в меню "Инструменты".

%center%%thumb width=550%Attach:main_menu.png%%

Аналогичное подменю появится в контекстном меню при щелчке правой кнопкой мыши на заметку.

%center%%thumb width=550%Attach:popup_menu.png%%

Плагин HackPage позволяет выполнять следующие действия:

* Менять идентификатор страниц.
* Устанавливать псевдоним для страницы.
* Менять папку хранения заметки без изменения имени заметки.

Рассмотрим эти возможности подробнее.

!!!! [[#changeuid]] Изменение идентификатора страницы

Начиная с версии OutWiker 1.8.0.732 появилась возможность делать перекрестные ссылки между страницами, используя не имена страниц, а их идентификаторы. Для получения ссылки на страницу предназначен пункт меню "Инструменты -> Копировать ссылку на страницу".

Первоначально идентификаторы выглядят примерно таким образом: {+[=page://__49ee3b83-dff4-479c-b1ad-a3edd2ace99e=]+}. Такие идентификаторы являются уникальными, но не очень наглядными для пользователей.

Если вы часто используете данную возможность, то у вас может возникнуть желание изменить этот идентификатор на что-то более осмысленное, например, на {+[=page://очень-полезная-страница=]+}.

При выборе пункта меню "HackPage - Изменить идентификатор страницы..." откроется диалог, в котором вы сможете изменить идентификатор страницы.

%center%Attach:dialog_change_page_uid.png


!!!! [[#setalias]] Установка псевдонима страницы

До OutWiker 2.0.0.808 имена заметок всегда совпадали с именем папки, в которой эти заметки расположены в файловой системе. Начиная с указанной версии страницы могут иметь псевдонимы, и тогда в интерфейсе программы они будут отображаться не по именам папок, а по псевдониму. В базовой версии программы нет интерфейса для установки псевдонимов. Это сделано специально, чтобы не путать неопытных пользователей.

С помощью плагина HackPage можно устанавливать псевдонимы страницы, для этого надо выбрать пункт меню "HackPage - Установить псевдоним страницы..." из контекстного меню или меню "Инструменты" в главном меню.

При этом будет показан диалог для ввода псевдонима.

%center%Attach:set_alias.png

Благодаря псевдонимам заметки могут иметь имена, включающие символы, недопустимые с точки зрения операционной системы, а также можно создать несколько заметок с одинаковыми отображаемыми именами внутри одной родительской заметки.

%center%Attach:notes_with_alias.png


!!!! [[#changefolder]] Изменение папки хранения заметки

Пункт меню "HackPage - Изменить папку страницы" позволяет переименовать папку, где хранится заметка, но при этом установить псевдоним, соответствующий исходному имени папки. Таким образом, с точки зрения пользователя заметка не меняет своего имени, но папка, где хранится заметка, переименовывается.

После выбора пункта меню "HackPage - Изменить папку страницы..." будет показан диалог для ввода нового имени папки

%center%Attach:change_folder_name.png

При вводе нового имени папки нужно учитывать, что на имя накладываются все те же ограничения операционной системы, что и на имена заметок. То есть, например, под Windows нельзя использовать такие символы как кавычки, звездочка, двоеточие и т.д. Под Linux нельзя использовать только прямые и обратные слеши.
