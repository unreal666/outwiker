!!! Using the Markdown plug-in

After installing the plug-in will be added to page creation dialog the new page type - "Markdown page". In all other Markdown-page similar to the built-in wiki page. It also has tabs for editing code, show result and generated HTML code.

%center%Attach:dialog.png


The toolbar has the buttons to insert a Markdown elements: headers, links, text emphasis with bold and / or italic, etc.
