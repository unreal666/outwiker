Плагин WebPage предназначен для сохранения страниц, скачанных из интернета. Такие web-страницы напоминают встроенные HTML-страницы и тоже позволяют редактировать скачанный HTML-код. Все картинки, стили CSS и скрипты JavaScript сохраняются в папке web-страницы, и для ее корректного отображения не требуется подключения к интернету.

!!! Использование плагина WebPage

!!!! Создание web-страниц

Web-страницы нельзя создать из стандартного диалога создания страниц, после установки плагина будет добавлен пункт меню ''Web-страница'', содержащий пункты для создания web-страниц.

%center%Attach:menu_ru.png

Аналогичные действия можно выполнить с помощью новых кнопок на панели инструментов над деревом заметок:

%center%Attach:tree_buttons_ru.png

После нажатия кнопки создания web-страницы откроется диалог, в котором нужно будет ввести ссылку на скачиваемую страницу и, при желании, выбрать метки для нее. После нажатия кнопки ''Ok'' начнется процесс скачивания страницы.

%center%Attach:dialog_download_progress_ru.png

Если все пройдет успешно, созданная web-страница появится в дереве заметок.


!!!! Редактирование web-страниц

Внешне web-страницы напоминают встроенные в OutWiker HTML-страницы. Вы также можете переключаться между режимами просмотра и HTML, редактировать исходный текст страницы, но web-страницы имеют дополнительную кнопку-переключатель (по умолчанию нажатую), позволяющую отключать выполнение скриптов на странице.

%center%Attach:disable_script_button_ru.png

Если страница без скриптов отображается некорректно, этот режим можно отключить. При нажатии кнопки ''Отключить скрипты на странице'' исходный текст страницы не меняется, скрипты удаляются только для отображения страницы в режиме ''Просмотр''.


!!!! Другие возможности

При выборе web-страницы меню ''Web-страница'' дополняется новыми пунктами.

%center%Attach:menu_full_ru.png

С помощью пунктов этого меню можно открыть исходную ссылку web-страницы в браузере, скопировать ссылку в буфер обмена или посмотреть лог скачивания страницы.
