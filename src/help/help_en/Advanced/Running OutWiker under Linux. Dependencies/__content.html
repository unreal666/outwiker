The following packages are required to run OutWiker on Linux:

  • python2.7
  • python-gtk2
  • python-wxgtk3.0
  • python-wxgtk-webview3.0
  • python-wxversion
  • python-webkit-dev
  • libwxgtk-webview3.0-0v5
  • python-pil or python-imaging
  • python-enchant
  • python-appindicator
  • ibus-gtk3
