The following packages are required to run OutWiker on Linux:
  • python (version 2.6 or 2.7)
  • python-wxgtk2.8
  • python-wxversion
  • python-webkit-dev
  • libgnomeprint2.2-0, libgnomeprintui2.2-0 (Note: these packages are needed for correct printing of text pages, wiki notations or original source pages containing Cyrillic characters. If text contains Latin characters only, then the packages are not necessary)
  • mimetex (this package is needed to render math formulae)
