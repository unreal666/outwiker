The program is an open source software and is distributed under the GPL. Sources are located at github.

To get the latest version of the source code from the git repository, execute the following command:
git clone https://github.com/Jenyay/outwiker.git

To get the latest stable version of the source code from git repository, execute the following commands:
git clone https://github.com/Jenyay/outwiker.git
cd outwiker
git checkout stable

Git must be installed to be able to execute these command.

To compile OutWiker from source code, execute following command (in the src directory):
python runoutwiker.py

Python 2.7 and wxPython 3.0.2 must be installed to compile OutWiker from the source code.
