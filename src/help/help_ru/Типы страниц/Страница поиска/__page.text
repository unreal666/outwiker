Страница глобального поиска по дереву заметок - это тоже отдельный тип страниц. Таких страниц может быть несколько. Это удобно, если часто требуется искать одни и те же фразы, тогда для каждой такой фразы можно сделать свою страницу поиска, чтобы не набирать искомую фразу каждый раз.
