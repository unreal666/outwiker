Исходники программы открыты и распространяются под лицензией GPL. Исходники расположены на github.

Чтобы получить самую свежую версию исходников из репозитория, выполните команду

git clone https://github.com/Jenyay/outwiker.git

Для получения исходников последней стабильной версии выполните команду

git clone https://github.com/Jenyay/outwiker.git
cd outwiker
git checkout stable


Для выполнения этих команд должна быть установлена программа Git.

Для запуска программы из исходников должны быть установлены Python 2.7 и библиотека wxPython 2.8.

Для запуска программы OutWiker из исходников выполните команду (в директории src)

python runoutwiker.py

При запуске под Linux обратите внимание на необходимые пакеты
