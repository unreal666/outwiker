Другим типом страниц, поддерживаемых OutWiker'ом являются страницы в вики-нотации. На этих страницах по-прежнему можно использовать HTML, но в первую очередь эти страницы рассчитаны на синтаксис, напоминающий (но не точно копирующий) нотацию, которая используется в [[pmWiki -> http://pmwiki.org/]].

Синтаксис вики-нотации описывается во вложенных заметках.
